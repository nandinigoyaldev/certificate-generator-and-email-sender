# Contributing to Certificate Generator

First off, thank you for considering contributing to this project! It's people like you that make open source such a great community.

## How to Contribute

1. **Fork the repository** to your own GitHub account.
2. **Clone the project** to your local machine.
3. **Create a branch** locally with a succinct but descriptive name (`git checkout -b feature/my-new-feature`).
4. **Make your changes** to the code.
5. **Commit your changes** with descriptive commit messages.
6. **Push your changes** to your fork.
7. **Submit a Pull Request** to the `main` branch of this repository.

## Reporting Bugs
If you find a bug in the source code, you can help us by submitting an issue to our GitHub Repository. Even better, you can submit a Pull Request with a fix.

## Feature Requests
You can request a new feature by submitting an issue. 

Please be respectful and patient. Thank you for making this project better!
