Quick Start Guide

1. Environment Setup (Critical)

To enable the email sending functionality, you must configure your SMTP credentials. Create a .env file in the root directory or add these to your Vercel Environment Variables:

SMTP_SERVER=smtp.gmail.com
EMAIL_ACCOUNTS=your.email@gmail.com,another@gmail.com
EMAIL_PASSWORDS=your_app_password,another_password

Note: If using Gmail, you must generate an App Password. Regular passwords will not work.

2. Formatting your Data File

Upload an Excel (.xlsx) or CSV file. The system requires exactly two columns to function correctly:

NameEmail
Jane Doejane@example.com
John Smithjohn@example.com

3. Generating Certificates

You have two modes of operation:

  • Live Mode: Turn "Demo Mode" OFF. The system will generate a PDF for each participant and instantly email it to them using your SMTP credentials.
  • Demo Mode: Turn "Demo Mode" ON. The system will skip the emailing phase, process all certificates, and securely download a .zip file containing all the PDFs to your local machine.
